# 🐍 Curso Desenvolvedor Python - SENAI DF

Repositório criado para armazenar atividades, exercícios e projetos desenvolvidos durante o curso de **Desenvolvedor Python** no **SENAI DF**, sob orientação do professor **Kauã Fellipe Ferreira Guimarães Jorge**.

---

## 📚 Sobre o Curso

O curso tem como objetivo desenvolver competências em programação com Python, abrangendo desde conceitos básicos até análise de dados e desenvolvimento de projetos práticos.

---

## 🧠 Conteúdo Programático

### 🔹 1. Introdução à Programação em Python
- História e aplicações do Python  
- Comparação com outras linguagens  
- Vantagens e desvantagens  
- Configuração do ambiente (instalação, variáveis, IDEs, ambiente virtual e pip)  
- Sintaxe básica e tipos de dados  
- Estruturas de dados (listas, tuplas, conjuntos, dicionários)  
- Entrada e saída de dados  
- Controle de fluxo (if, elif, else, for, while, break, continue, pass)  

---

### 🔹 2. Estruturas de Dados e Funções
- Estruturas de dados avançadas  
- Métodos e mutabilidade  
- Funções (definição, parâmetros, escopo)  
- Funções lambda e funções de ordem superior (map, filter, reduce)  
- List Comprehensions  

---

### 🔹 3. Programação Orientada a Objetos (POO)
- Classes e objetos  
- Atributos e métodos  
- Construtores e destruidores  
- Herança e polimorfismo  
- Encapsulamento  
- Propriedades e decoradores  

---

### 🔹 4. Manipulação de Arquivos e Exceções
- Leitura e escrita de arquivos  
- Modos de arquivo (texto e binário)  
- Context manager (`with`)  
- Tratamento de exceções (try, except, else, finally)  
- Exceções personalizadas  

---

### 🔹 5. Bibliotecas Padrão e Módulos Externos
- Manipulação de datas (`datetime`, `time`)  
- Expressões regulares (`re`)  
- Requisições HTTP com `requests`  
- APIs (GET, POST, JSON)  
- Web scraping com BeautifulSoup  

---

### 🔹 6. Introdução a Dados e Análise
- Pandas (Series e DataFrame)  
- Leitura de dados (CSV, Excel)  
- Manipulação e limpeza de dados  
- Visualização com Matplotlib e Seaborn  
- Estatística básica  
- Introdução ao Machine Learning com Scikit-learn  

---

### 🔹 7. Desenvolvimento de Projeto
- Planejamento e definição de escopo  
- Desenvolvimento prático e iterativo  
- Testes e melhorias  
- Apresentação final do projeto  

---

## 🛠️ Tecnologias Utilizadas
- Python 3.x  
- VS Code / PyCharm  
- Git e GitHub  
- Bibliotecas: Pandas, Matplotlib, Seaborn, Requests, BeautifulSoup, Scikit-learn
